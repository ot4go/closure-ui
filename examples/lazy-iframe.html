closure-ui — <closure-lazy-iframe>

Self-contained demo (works from file://) of <closure-lazy-iframe>: a collapsible panel with a header (label + expand/collapse chevron) and a body hosting an <iframe> whose src is only assigned the first time the panel is expanded — until then no frame exists and the embedded page costs nothing. The frames here point at the sibling example pages, so no server is needed.

This page is deliberately pure HTML — it has no stylesheet at all, so everything you see is the component's built-in default styling. For restyling via CSS variables see theming.html.

1. Collapsed by default — expand to load

The panel starts collapsed (no expanded attribute). Open the network tab before expanding: buttons-actions.html is only requested on the first expand. Light-DOM children are slotted into the body as a placeholder and stay visible until the frame's first load. The iframe stretches to fill the body; without an explicit host height the body falls back to --lazy-iframe-height (320px).

The buttons demo page loads here the first time you expand the panel.

Programmatic control + events

Collapsing only hides the body — the loaded document stays alive, so re-expanding is instant and fires no lzi-load. unload() collapses and drops the frame: the next expand reloads src from scratch.

(lzi-toggle / lzi-load / lzi-loaded events will appear here)

2. Starts expanded (loads immediately)

With the expanded attribute in the markup the panel opens — and therefore loads — right away. The attribute is the single source of truth: server-rendered HTML can open a panel by simply including it, and expand() / collapse() / toggle() just set or remove it.

Markup

<closure-lazy-iframe label="Buttons &amp; actions demo"
                     src="buttons-actions.html"
                     iframe-title="Buttons and actions demo page">
  <p>Placeholder shown until the frame's first load.</p>
</closure-lazy-iframe>

<closure-lazy-iframe label="Theming demo" src="theming.html" expanded></closure-lazy-iframe>

<script>
  const panel = document.querySelector('closure-lazy-iframe');
  panel.addEventListener('lzi-toggle', e => console.log('expanded:', e.detail.expanded));
  panel.addEventListener('lzi-load',   e => console.log('loading:', e.detail.src)); // cancelable
  panel.addEventListener('lzi-loaded', e => console.log('ready:', e.detail.src));
  panel.expand();     // collapse() / toggle() / unload() also available
</script>
