closure-ui — Theming with CSS variables

The components style themselves; pages never need CSS to make them look right. When you want a different look, every component exposes CSS variables (listed per component in doc/closure-ui.md) that can be set on :root, on a wrapper, or on the element itself. This page demonstrates the technique — it is the only example page with a <style> block.

1. Shared design tokens

Default rendering (no CSS at all):

Inside a .corporate wrapper that redefines --primary, --border, --font… (focus a field to see the teal ring):

.corporate {
  --primary: #0f766e;
  --primary-light: #ccfbf1;
  --border: #99f6e4;
  --font: Georgia, serif;
  --text: #134e4a;
}

<div class="corporate">
  <credential-pwd name="p3"></credential-pwd>
</div>
